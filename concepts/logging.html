Logging

runEma’s action monad supports the MonadLoggerIO constraint, as defined by monad-logger. This means that you can use any of the logging functions from monad-logger to add logging to your application. monad-logger-extras is used to colorize the logs.

logInfoNS "myapp" "This is an info message"
logDebugNS "myapp" "This is a debug message"
